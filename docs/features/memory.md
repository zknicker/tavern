# Memory

Memory is the small, durable set of facts and preferences agents keep in mind.
It is not a document store and not a general wiki.

## In the box

* **Inspectable memory.** Users can see what agents remember.
* **Editable records.** Bad or stale memory can be changed or removed.
* **Agent-aware context.** Memory can shape later turns without hiding from the
  operator.
* **Clear boundary.** Big documents and working notes belong in the
  knowledgebase.
