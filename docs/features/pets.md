# Pets and rewards

Pets and rewards make long-running agentic work feel less sterile.

## In the box

* **Companion presence.** A small persistent character can make the app feel
  alive.
* **Progress rewards.** Completed work can create small moments of delight.
* **Low-stakes fun.** The layer never obscures work, settings, or errors.
