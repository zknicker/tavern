# Knowledgebase

The knowledgebase is Tavern's wiki-style workspace for durable working material.
It is where agents and users can keep notes, files, citations, and structured
project context.

## In the box

* **Wiki-style pages.** Durable pages with titles, body content, and links.
* **Backlinks and collections.** Related notes are easy to discover.
* **Files and citations.** Source material stays attached to the work it
  supports.
* **Agent-authored notes.** Agents can create working material without turning
  every note into long-term memory.
* **Project context.** Pages can become the shared brain for a business,
  workflow, or research thread.
